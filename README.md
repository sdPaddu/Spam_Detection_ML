# Spam Detection API 🚀

## 📌 Project Overview
This project is a Machine Learning-based Spam Detection system built using FastAPI.

## ⚙️ Tech Stack
- Python
- Scikit-learn
- TF-IDF
- FastAPI

## 🚀 Features
- Predicts whether a message is Spam or Not
- REST API using FastAPI
- Trained ML model (Logistic Regression / Naive Bayes)

## 📂 Project Structure
- app/ → FastAPI code
- model/ → trained ML model
- dataset/ → dataset used

## ▶️ How to Run
```bash
uvicorn app:app --reload
